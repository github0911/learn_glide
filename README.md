### 如何使用glide
[参考示例](https://blog.csdn.net/guolin_blog/article/details/53759439)
- 引入gradle配置
```
repositories {
  mavenCentral()
  google()
}

dependencies {
  implementation 'com.github.bumptech.glide:glide:4.9.0'
  annotationProcessor 'com.github.bumptech.glide:compiler:4.9.0'
}
```
